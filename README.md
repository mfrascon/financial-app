## AI Finance Platform

A full-stack AI-powered finance tracking application built with modern web technologies. This app allows users to manage expenses, track financial activity, and automatically extract data from receipts using AI.

🚀 Features
📊 Expense Tracking
Add, edit, and delete expenses
Categorize transactions
View spending history
🤖 AI Receipt Scanning
Upload receipts
Automatically extract:
Amount
Date
Merchant
Category (optional)
Auto-fill expense entries
🔐 Authentication
Secure user authentication and session management
📈 Financial Insights
Track total spending
Analyze expenses by category
Monthly summaries
⚡ Event-Driven Workflows
Background processing for AI tasks and automation
🛡️ Security & Rate Limiting
API protection and abuse prevention
🧰 Tech Stack
Frontend
React 19
Next.js 15 (App Router)
Tailwind CSS
Shadcn UI
Backend
Next.js API Routes / Server Actions
PostgreSQL (via Supabase)
Prisma ORM
Authentication
Clerk
AI & Background Jobs
Inngest (event-driven workflows)
Security
Arcjet (rate limiting & bot protection)
